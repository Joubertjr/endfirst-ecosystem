## SOFTWARES PRINCIPAIS IDENTIFICADOS

### **1. Zotero** (Open Source)
- Gratuito + Open Source (AGPL)
- Multi-plataforma (Windows, Mac, Linux)
- 300 MB storage grátis
- Conectores para Firefox, Chrome, Safari
- Acesso web + desktop + cloud

### **2. Mendeley** (Elsevier)
- Gratuito até 2 GB
- Multi-plataforma
- Sincronização automática
- Usável offline
- Proprietary com API pública

### **3. EndNote** (Clarivate)
- US$ 299.95 (pago)
- Versão web gratuita (EndNote basic)
- Mais antigo (1988)
- Proprietary

### **4. JabRef** (Open Source)
- Gratuito + Open Source (MIT)
- Java-based
- BibTeX e BibLaTeX manager
- Multi-plataforma

### **5. Citavi**
- US$ 89-1947
- Dados locais ou cloud
- Busca de databases direto da interface
- Proprietary

### **6. RefWorks**
- Assinatura institucional
- Web-based
- Integração Word & Google Docs
- Proprietary

### **7. Paperpile**
- US$ 2.99/mês (acadêmicos)
- Web-app
- Integração Google Docs
- Chrome only

### **8. Papers** (ReadCube)
- US$ 3-5/mês
- Web + Desktop + Mobile
- Word & Google Docs add-in
- Browser extension

## GAPS IDENTIFICADOS (O QUE FALTA)

### **Gap 1: Foco em Artigos Acadêmicos**
- Maioria focada em papers científicos
- Pouco suporte para outros tipos de fontes
- Limitado para conteúdo web, vídeos, podcasts

### **Gap 2: Complexidade**
- Curva de aprendizado alta
- Muitas funcionalidades desnecessárias para uso simples
- Interface complexa

### **Gap 3: Custo**
- Soluções profissionais são caras (US$ 299+)
- Soluções gratuitas têm limites de storage
- Assinaturas mensais/anuais

### **Gap 4: Dependência de Plataforma**
- Alguns só funcionam em navegadores específicos
- Alguns só funcionam em sistemas operacionais específicos
- Lock-in de vendor

### **Gap 5: Foco em Citação Formal**
- Muito foco em gerar bibliografias formatadas
- Pouco foco em gestão de conhecimento
- Pouco foco em descoberta de insights

## FERRAMENTAS PKM (PERSONAL KNOWLEDGE MANAGEMENT)

### **Diferença entre Citation Management e PKM:**

**Citation Management (Zotero, Mendeley, EndNote):**
- Foco em **citações formais**
- Gerar bibliografias formatadas
- Armazenar PDFs de papers
- Organização por pastas/tags

**PKM (Obsidian, Roam, Notion):**
- Foco em **gestão de conhecimento**
- Conexões entre ideias (backlinks)
- Descoberta de insights
- Pensamento em rede

## INSIGHTS PARA BANCO DE REFERÊNCIAS

### **Insight 1: Conexões > Armazenamento**
- PKM foca em **conectar** ideias, não só armazenar
- Backlinks revelam relações entre fontes
- Graph view mostra rede de conhecimento

### **Insight 2: Gap Analysis é Crítico**
- Identificar **lacunas** no conhecimento
- Descobrir o que falta pesquisar
- Guiar pesquisas futuras

### **Insight 3: Local-First é Valorizado**
- Usuários querem **controle** dos dados
- Privacidade é importante
- Markdown é formato preferido (portável)

### **Insight 4: Extensibilidade é Essencial**
- Plugins permitem customização
- Integrações com outras ferramentas
- API para automação

### **Insight 5: Análise Visual é Poderosa**
- Graph view ajuda a entender estrutura
- Network visualization revela padrões
- Clusters mostram temas

## GAPS IDENTIFICADOS (PKM)

### **Gap 1: Foco em Notas, Não em Citações**
- PKM não gerencia citações formais
- Não gera bibliografias
- Não valida fontes

### **Gap 2: Falta de Validação Científica**
- Não usa Hierarquia de Evidências
- Não classifica qualidade de fontes
- Não rastreia fontes primárias

### **Gap 3: Integração Citation + PKM**
- Ferramentas separadas (Zotero + Obsidian)
- Workflow fragmentado
- Duplicação de esforço

## CONCLUSÃO DA PESQUISA

**O QUE EXISTE:**
1. **Citation Management:** Zotero, Mendeley, EndNote (foco em citações formais)
2. **PKM:** Obsidian, Roam, Notion (foco em conexões entre ideias)
3. **Análise Avançada:** InfraNodus (gap analysis, network visualization)

**O QUE FALTA:**
1. **Integração** entre citation management e PKM
2. **Validação científica** (Hierarquia de Evidências)
3. **Base dinâmica e crescente** otimizada
4. **Gap analysis** para fontes científicas

**OPORTUNIDADE:**
Criar Banco de Referências que **combina**:
- Citation management (rastreabilidade)
- PKM (conexões e insights)
- Validação científica (Hierarquia de Evidências)
- Gap analysis (identificar lacunas)
- Base dinâmica (crescimento contínuo)
