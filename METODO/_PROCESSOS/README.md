# METODO/_PROCESSOS — Artefatos comuns (não-persona)

Este diretório contém artefatos **comuns a todas as personas**, que **não** são personas.

## O que entra aqui

- Contratos, vínculos e mapeamentos de processo que são usados por múltiplas personas.
- Artefatos que o template/auditoria de personas não deve interpretar como “persona”.

## Diretórios

- `VINCULOS_PROCESSO/`: vínculos papel ↔ fase / tipo de demanda / tipo de produto.

## Regra estrutural

- **Não colocar personas aqui.**
- **Não colocar processos comuns dentro de `METODO/PERSONAS/`.**
